Spring-boot with jdbc Coding Example (Work In Progress)

Explanation:

This project implements Spring-Boot Framework. 
This application is capable of performing the basic Create, Retrieve, Update and Delete (CRUD) operations of Insurance-Policy data into an mysql database. 
Following implementations can be executed:
	<hostName>:<portNumber>/addNewPolicy
	<hostName>:<portNumber>/getPolicyByID
	<hostName>:<portNumber>/getAllPolicies
	<hostName>:<portNumber>/deletePolicyByID
	<hostName>:<portNumber>/deleteAllPolicies

Languages/Frameworks used: Java, Spring-Boot, spring-jdbc(to-be-done), MySQL,Maven, JSP
